# CanteenOrderSystem

## 介绍

**本项目是我学校的生产实习项目**

理工二食堂某窗口的订餐系统，为莘莘学子提供二食堂某窗口订餐服务，可以派送到寝室、教室、实验室等地点，不必出门，也能享受二食堂美食。



## 技术架构

<img src="https://kkbank.oss-cn-qingdao.aliyuncs.com/note-img/image-20221106185646994.png" alt="image-20221106185646994" style="zoom:150%;" />

前端：

- Vue
- TS
- Vuex
- ElementUI

后端：

- springboot
- MySQL
- Redis
- MyBatisPlus
- Druid
- EasyExcel
